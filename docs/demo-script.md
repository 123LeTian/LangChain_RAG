# Demo Script

## Setup

```bash
pip install -r requirements.txt
python -m src.api.app
```

## Demo Flow

1. **Upload documents** — Upload sample PDFs/Markdown files
2. **Query knowledge base** — Ask questions against the indexed documents
3. **Graph exploration** — Visualize entity relationships
4. **Agent reasoning** — Multi-step complex queries
5. **Evaluation** — Run metrics and view reports

## Sample Queries

- "What are the key findings in the uploaded reports?"
- "How do entities X and Y relate to each other?"
- "Summarize the main topics across all documents."
