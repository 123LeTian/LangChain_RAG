## Metrics

### Retrieval Quality
- MRR (Mean Reciprocal Rank)
- NDCG@k (Normalized Discounted Cumulative Gain)
- Recall@k
- Precision@k

### Answer Quality
- Faithfulness (hallucination detection)
- Answer Relevance
- Context Relevance

### Agent Quality
- Tool Selection Accuracy
- Path Efficiency
- Task Success Rate

## Datasets

Evaluation datasets are stored under `datasets/evaluation/`.
Demo documents for development are under `datasets/demo_docs/`.
